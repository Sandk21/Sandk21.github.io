Application de détection de faux billets

Développement d'une application web pour détecter les vrais des faux billets de banques pour l'Organisation Nationale de Lutte contre le Faux-Monnayage (ONCFM), organisme public, ayant pour mission de développer des méthodes avancées d’identification des contrefaçons de billets en euros.

Développement d'un algorithme de machine learning capable de distinguer les billets authentiques des contrefaçons en s'appuyant sur leurs caractéristiques géométriques.
